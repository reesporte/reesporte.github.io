Determining Syntactic Features of English Registers with Linear Support Vector Machine
An experimental project for a computational linguistics course.

I worked on a group project for my computational linguistics class trying to identify the most useful syntactic features for training a linear SVC machine to classify between academic, conversational, and news English registers. You can find the source on Github. The poster is available for download here.
